# LearnLM















LearnLM is no longer a separate listing in AI Studio. Instead, LearnLM
capabilities have been integrated into Gemini starting with the
[2.5 model series](/gemini-api/docs/models).

To learn more about LearnLM's current status, visit the [LearnLM](https://cloud.google.com/solutions/learnlm) documentation.
